# Описание проекта
Приложение представляет из себя веб-интерфейс для игры в "Словотрон 9000" с интеграцией с Twitch.
С помощью открытого API (api.contextno.com) и сайта контекстно.рф сначала запрашивается UUID секретного слова, а затем проверяются слова-отгадки, которые пользователи отправляют в чат Twitch.


# Технологический стек
В проекте не используются никакие фреймворки, только нативный HTML, CSS и JavaScript


# Дополнительные библиотеки
- tmi - взаимодействие с твичом
- confetti - эффекты конфети


# Важные концепции и константы
- **Дистанция (Distance):** Чем меньше число, тем ближе проверочное слово к загаданному. 1 — победа.


# Деплой и ветки
- **`dev`**: Все доработки и фичи вносятся сюда.
- **`main`**: Стабильная версия, автоматически деплоится на GitHub Pages через GitHub Actions.
